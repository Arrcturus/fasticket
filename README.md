# Fasticket - Módulo de Venta de Entradas para Odoo

## Descripción
Fasticket es un módulo de Odoo diseñado para gestionar la venta de entradas para eventos, similar a plataformas como Ticketmaster. Permite a los usuarios crear eventos, configurar diferentes tipos de entradas, gestionar la disponibilidad, y procesar la compra de boletos en línea.

## Características
- Creación y gestión de eventos (aún no implementado)
- Configuración de múltiples tipos de entradas con precios diferentes (aún no implementado)
- Generación de códigos QR/códigos de barras para validación de entradas (aún no implementado)
- Proceso de venta de entradas integrado con el sistema de e-commerce (aún no implementado)
- Panel de administración para gestionar ventas y estadísticas (aún no implementado)
- Informes detallados de ventas por evento (aún no implementado)

## Instalación (aún no implementado)
1. Clone este repositorio en su carpeta de addons de Odoo:
    ```bash
    git clone https://github.com/su-usuario/fasticket.git /mnt/extra-addons/fasticket
    ```
2. Actualice la lista de aplicaciones en Odoo
3. Busque e instale el módulo "Fasticket"

## Configuración (aún no implementado)
Después de instalar el módulo: (aún no implementado)
1. Vaya a Configuración > Fasticket > Ajustes (aún no implementado)
2. Configure los parámetros generales para la venta de entradas (aún no implementado)
3. Cree sus primeros eventos en Fasticket > Eventos (aún no implementado)

## Dependencias
- `website_sale`: Para la integración con el carrito de compras
- `mail`: Para notificaciones por correo electrónico
- `report`: Para la generación de entradas en PDF
